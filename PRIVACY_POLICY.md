# EdgeTranslate333 隱私權政策與服務條款

## 1. 簡介

歡迎使用 **EdgeTranslate333**。我們致力於提供高效、簡潔且安全的網頁翻譯體驗。本文件說明我們如何處理您的資訊，以及您使用本擴充功能的相關規範。

## 2. 資料蒐集

我們遵循最小蒐集原則：

- 不蒐集個人資料（如姓名、電子郵件、電話等）。
- 不進行追蹤，不記錄瀏覽行為。
- 不出售或分享任何使用者資料。

## 3. 權限與用途

為了讓功能正常運作，EdgeTranslate333 會使用下列權限：

- **網站存取**：用於在網頁上顯示翻譯結果與側邊面板。
- **瀏覽器儲存（Sync Storage）**：用於保存您的偏好設定（如主題、語言等）。
- **文字轉語音（TTS）**：使用瀏覽器內建 API 朗讀文字，不會另外傳送音訊資料。

## 4. 第三方服務

當您主動觸發翻譯時，系統會將您選取的文字送至您選擇的翻譯服務（如 Google 翻譯、Bing 翻譯等）。

- 僅在您主動觸發翻譯時才會傳送必要文字。
- 該行為受各翻譯服務的隱私政策約束。

## 5. 使用者控制

- 您可隨時在設定中修改偏好。
- 移除擴充功能將會清除本地與同步儲存的設定。

## 6. 安全性

所有翻譯請求皆使用 HTTPS 進行傳輸。我們不會建立或保存您的內容副本。

## 7. 服務條款

- **使用規範**：僅可合法使用本工具，不得用於任何違法或侵權行為。
- **免責聲明**：本工具按現狀提供，不保證翻譯結果的準確性或即時性。
- **責任限制**：在法律允許範圍內，本工具不對任何直接或間接損害負責。
- **更新**：我們可能隨時更新功能或調整條款。

## 8. 變更

若政策或條款有重大變更，將隨版本更新公告。
